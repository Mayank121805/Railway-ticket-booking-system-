## Project Overview
This project is a **Railway Ticket Booking System** developed using **C programming with CGI**, along with **HTML and CSS** for the frontend.  
The system simulates basic railway booking operations and is intended for **academic and learning purposes**.

## Technologies Used
- C Programming (CGI)
- HTML5
- CSS3
- Apache Web Server
- Text files for database storage
